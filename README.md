# Skills Summary

## cadastral-vzd

> **Note:** Do not use this skill in production. It was created partially for testing skill capabilities in [claude.ai](https://claude.ai/).

This skill was created for the Datu Skola [presentation](https://www.datuskola.lv/index.php/2025/12/01/ka-maksligajam-intelektam-paredzet-nakotnes-kadastra-vertibas/) on predicting future cadastral values with AI.

**Excel compatibility**  
If this skill doesn't work in Excel (using Claude plug-in) due to the `.parquet` file included in the skill, you can use this [MCP server](https://aivis-hf-vzd-mcp-test.hf.space/gradio_api/mcp/) which extracts base value for land and premise group. It's hosted on my HuggingFace profile as the [hf-vzd-mcp-test](https://huggingface.co/spaces/Aivis/hf-vzd-mcp-test) space.
